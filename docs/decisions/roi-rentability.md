# Decisión — ROI / Rentabilidad

## Decisión

SumiControl debe incluir ROI como métrica transversal, no como un reporte aislado.

## Ubicaciones

- Dashboard owner/admin: bloque Rentabilidad / ROI.
- Reportes: ROI / Rentabilidad.
- Productos e inventario: ROI por producto, categoría, variante y rotación.
- Compras: ROI por orden de compra, proveedor y lote recibido.
- Cilindros y recargas: ROI de recargas e impacto de cilindros pendientes/perdidos.
- Clientes: ROI comercial, utilidad generada y riesgo de crédito.
- Matrices administrativas: matriz ROI mensual/anual.

## Fórmula base

```text
ROI = (Utilidad neta / Inversión o costo) x 100
```

## MVP

Para la demo, el ROI se calculará con la data disponible de ventas, compras, costos, utilidad y matrices 2024.
